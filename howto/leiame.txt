Esta pasta contém uma versão do Howto em Português(Brasil). Ele
contem parte do conteúdo do original mas não é apenas uma tradução,
pois contém as experiencias do grupo que traduz o manual, assim
como as nossas próprias definições e regras.

Para que você possa gerar este manual você deverá:
1. Executar o arquivo install.bash, que irá copiar os arquivos necessários e
criar uma pasta que guardará o resultado(arquivos html)
2. Na pasta raíz(aquela que tem o configure) execute:
./configure --with-lang=pt_BR
make -f howtolocal

Apenas isto. Caso você queira gerar novamente o html terá que apagar o arquivo index.html

Problemas a resolver:
    Fazer uma faxina no howtolocal.in deixando apenas o necessário para a geração do manual.
    Resolver a questão da necessidade de apagar o index.html antes de gerar novamente o manual.

Descrição dos arquivos:
    arquivos.ent: contém as entidades para os arquivos. Deve se adicionar uma nova entidade
                  para cada arquivo adicionado no howto. Depois colocar esta entidade aonde
                  o arquivo deve ser inserido.
    howto.ent: aqui estão as nossas entidades assim como aquelas que existem no howto original.
               Manter urls e emails como entidades para facilitar alterações.
    howto.xml: este é o início do howto.
